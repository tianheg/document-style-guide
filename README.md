# 文档写作规范

如何写出优秀的文档？我会写作和收集这些规范。

## 小技巧

- 文件命名：英文无大小写要求;
- 文件命名：只允许使用一个 `-` 作为间隔
- 文件命名：文件名只能是英文
- 当中文作为辅助注释时，要用全角括号括起来`（）`，而不是半角括号`()`
- 使用 Vuepress 时，可以用 `<url>` 代替 `[url](url)`
- 只放纯链接，不放文本，例：`<url>`，链接中的中文字符使用中文输入，不采用编码后的很长的书写方式
